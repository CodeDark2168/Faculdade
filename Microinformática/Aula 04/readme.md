## Calculadora IMC

- Crie um programa em C# com base no layout apresentado.
- Ao calcular o IMC, deverá ser adicionado o Nome, IMC e categoria que a pessoa se encontra.
    - IMC = Peso / Altura²
    - Abaixo do peso | < 18.5
    - Normal | >= 18.5 e < 25
    - Sobrepeso | >= 25 e < 30
    - Obesidade | >= 30
